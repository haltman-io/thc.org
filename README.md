please dont ask me where we found the images...  
* some came from PUBLIC x/twitter profiles of THC;
* some from PUBLIC mastodon/infosec.exchange; 
* some from PUBLIC profiles of thc friends like phrack and other people that was PUBLICLY mentioned in THC PUBLIC posts
* others came from the wayback machine on the THC domains (thc.org, segfault.net and others)
